# PnPDataset Project Documentation

## 项目概述 (Project Overview)
本项目旨在对17-18世纪欧洲艺术史索引数据进行全面的清洗、标准化与重构。通过从原始PDF/Markdown提取数据，经过多轮清洗、实体对齐（Wikidata）及LLM增强，最终生成高质量的结构化数据集。

## 目录结构说明 (Directory Structure)

### 原始数据层 (Raw Data Layer)
*   **00-book/**: 存放原始书籍资源（如PDF扫描件）。
*   **01-Index-PDF/**: 索引部分的PDF拆分文件。
*   **02-Index-MD/**: 从PDF转换得到的Markdown格式索引文件（按字母分类）。
*   **03-Index-CSV/**: 从Markdown解析出的原始CSV数据文件。

### 数据处理层 (Processing Layer)
*   **04-Index-Enrich/**: 初步清洗与增强后的CSV文件（按字母分类），包含基础的格式修正。
*   **05-HandmadeDataset/**: 人工构建的参考数据集，用于实体对齐与校验。
    *   `name-English_table.csv`: 人名实体映射表。
    *   `gio-English_table.csv`: 地名/组织实体映射表。
    *   `work-English_table.csv`: 作品实体映射表。
*   **06-EntityMerge/**: 实体合并阶段的中间文件。
*   **07-LLM-Enhancement/**: LLM（大语言模型）批量处理结果及相关文档。
    *   `*.csv`: 分批次处理的数据文件。
    *   `Reademe-A.md` / `Readme-B.md`: 技术执行报告与规格说明书。

### 结果数据层 (Result Data Layer)
*   **08-Data-Remerge/**: **[当前核心数据]** 经过最终合并、去重与校验的数据集。
    *   `01-Merged_Recheck.csv`: 合并后的全量待检查数据。
    *   `02-Merged_Recheck_Deduplicated.csv`: 去重后的完整数据集（保留了变体信息）。
    *   `03-Merged_Recheck_Simplified.xlsx`: **[最终交付文件]** 简化版Excel数据集，仅包含标准名、分类与备注。
    *   `Duplicate_Rows_Report.txt`: 重复行审计报告。
    *   `Deduplication_Audit_Report.txt`: 去重逻辑审计报告。

### 辅助层 (Auxiliary Layer)
*   **09-QID-Crosscheck/**: 用于存放QID（Wikidata ID）交叉验证的临时文件（目前为空）。
*   **Process-Python/**: 项目使用的所有Python处理脚本。
    *   包含从数据提取、清洗、分析到去重的全套工具链。
    *   关键脚本：`28_List_All_Duplicate_Rows.py`, `29_Deduplicate_Recheck.py`, `33_Convert_To_Excel_And_Audit.py` 等。
*   **BU-Crosscheck/** & **BU-UNMatched/**: 备份与未匹配数据存档。
